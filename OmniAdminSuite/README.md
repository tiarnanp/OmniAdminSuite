# OmniAdminSuite

A Spigot/Paper admin GUI plugin for Minecraft 1.21.x.

## Commands
- `/oas` — Open the main admin GUI
- `/oasreload` — Reload the plugin

## Permissions
- `oas.use` — Full access (default: OP)

## Building
This project uses GitHub Actions to auto-compile. Just push to `main` and download the `.jar` from the **Actions** tab.
